# VibeTunes

VibeTunes est une application musicale moderne et immersive inspirée des plateformes premium de streaming audio comme Spotify, Apple Music et Deezer.

L’objectif du projet est de créer une expérience utilisateur élégante, fluide et responsive permettant d’écouter des previews musicales, explorer des playlists tendances et naviguer dans une interface haut de gamme entièrement développée en HTML, CSS et JavaScript vanilla.

---

# Conception Assistée par IA

La conception et le développement initial du projet ont été réalisés avec l’aide d’outils d’intelligence artificielle modernes afin d’accélérer le prototypage, améliorer la qualité du design UI/UX et optimiser l’architecture frontend.

Outils IA utilisés :

- Cursor
- Manus
- Lovable

Ces outils ont été utilisés pour :

- Génération de l’interface utilisateur
- Structuration du frontend
- Création des composants UI
- Optimisation du design responsive
- Assistance au développement JavaScript
- Accélération du prototypage
- Amélioration de l’expérience utilisateur

---

# Spécification du Projet

## Objectif

Développer une application frontend de streaming musical moderne avec :

- Une interface premium responsive
- Un lecteur audio interactif
- Une intégration future avec l’API Deezer
- Une expérience utilisateur immersive
- Un design dark/light moderne
